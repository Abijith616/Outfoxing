# Outfoxing - MCA Job Placement Platform

## Project Origin

This project was born out of the necessity to avoid the placement cell hibernation state in MCA colleges. The traditional placement cells often become inactive or ineffective, leaving MCA students in a state of uncertainty about their career prospects.

## Problem Statement

- **Placement Cell Hibernation**: Many MCA college placement cells become inactive or ineffective
- **Job Opportunity Gap**: MCA students lack dedicated platforms for job opportunities
- **Career Uncertainty**: Students face uncertainty about placement prospects and career paths

## Solution

Outfoxing is a dedicated job opening platform specifically designed for MCA students to:

- 🔍 **Discover Opportunities**: Access curated job openings relevant to MCA graduates
- 🎯 **Targeted Matches**: Get matched with companies seeking MCA talent
- 📊 **Career Insights**: Track placement trends and market demands
- 🚀 **Skill Development**: Identify in-demand skills for better placement prospects

## Features

- **Job Aggregation**: Collects MCA-specific job openings from various sources
- **Company Profiles**: Detailed information about companies hiring MCA graduates
- **Skill Assessment**: Tools to evaluate and improve technical skills
- **Placement Analytics**: Data-driven insights into placement trends
- **Community Forum**: Platform for students to share experiences and tips

## Technology Stack

- **Frontend**: React Type Script
- **Backend**: Python FAST API
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Secure user management
- **Deployment**: Cloud-based hosting


### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/outfoxing.git
cd outfoxing
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
# Edit .env with your Supabase credentials
```

4. Run the development server
```bash
npm run dev
```
